All activities in this module relate to transferable skills, employability and gaining network development
skills. Students will have opportunities to develop skills such as
• Web development and scripting
• Problem solving
• Time management
• Analytical skills
and achieve relevant professional certifications

Rationale

This subject introduces students to the fundamental concepts and models of Web application
development. Students will be equipped with the skills to create platform independent and responsive
web applications. The technologies covered in the module include HTML, CSS, JavaScript, PHP and
ASP.NET. Students will aim to create a portfolio of webpages that demonstrate a range of HTML and
CSS features. Students will also aim to develop a JavaScript application in response to a specified
problem and implement web security. By the end of the module students would be able to analyse
the extent to which a web application meets the criteria defined for its current use and future
development.

Indicative Content

The indicative content of the module may include the following:
• Problem solving and decision-making techniques
• Programming concepts
• Concepts of responsive designs and platform independence
• HCI principles and universal accessibility
• Cascading Style Sheet (CSS) Basics
• Client-side scripting such as JS, CSS and HTML
• Graphics & Text Styling Basics
• Media & Interactivity Basics
• PHP, PHP Framework and how PHP interacts with databases
• Introduction to ASP.NET.
• Processing forms with PHP
• Website security: content security policies, headers, cookies etc.
