# COM4012M – Web Programming  
## 12-Week Lecture Syllabus (First Year Computer Science)

### Module Context

This module builds on students’ prior experience of introductory programming in Python and introduces the principles and practice of modern web application development. Students will develop platform-independent, responsive, and secure web applications using front-end and back-end technologies including HTML, CSS, JavaScript, PHP, and introductory ASP.NET concepts.

The module aligns with the specification emphasis on:

* Web development and scripting
* Problem solving and decision-making
* Responsive design and platform independence
* HCI principles and accessibility
* Client-side and server-side programming
* Form processing and database interaction
* Web security and professional practice

# Teaching Approach

Each week should include:

* Lecture theory and demonstration
* Guided lab exercises
* Incremental portfolio development
* GitHub usage for version control and issue tracking
* Reflection on professional practice

This creates strong continuity from introductory Python programming into applied web development.
